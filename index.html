NAVAJO WORD PROCESSOR
The Navajo Word Processor is a word processor that allows users to type documents in the Navajo Language. The word processor was developed by San Juan College students. My class worked on the existing word processor, developed by previous classes, to fix bugs and add features. A feature I created and prototyped was a Navajo to English Dictionary. I thought this would be very useful to users not fluent in Navajo. The dictionary was design to simply display the English words and the corresponding Navajo phrase. This feature became a major marketing point for the word processor and later classes refined the feature to offer more options to the user.
